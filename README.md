# Jasper artbox

*创建：2026-07-04*

给 Jasper（6 岁）存画画照片的应用。手绘涂鸦风格 UI（Gaegu + Permanent Marker 字体，粗黑描边贴纸感），iPad 横屏布局，包含四个界面：

1. **Gallery** — 画作画廊，侧边栏导航（Gallery / Wall / Favorites / + New Art），网格卡片带作品名、拍摄地点（新加坡/上海等）和日期
2. **Capture** — 拍照录入新画作
3. **Detail** — 单幅画详情
4. **Painting Wall** — 画作墙展示
